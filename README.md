# Employee Payroll App

A simple Java console application developed to understand and practice Object-Oriented Programming (OOP) concepts through an Employee Payroll Management System.

## Features

- Employee Registration
- Employee Login & Authentication
- Payslip Generation
- Payslip Print / Download
- Employee Dashboard
- Input Validation using Regular Expressions

## OOP Concepts Used

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Composition
- Aggregation
- Interfaces
- Exception Handling
- Custom Exceptions
- Factory Pattern
- Stream API

## Use Cases

### UC1 - Employee Registration
- Register a new employee
- Validate employee details
- Store employee information

### UC2 - Employee Authentication & Login
- User login
- Password hashing
- Session management
- Role-based authentication

### UC3 - Payslip Generation
- Calculate salary components
- Generate monthly payslip
- Calculate deductions and net salary

### UC4 - Payslip Print / Download
- Clone payslip
- Save payslip as Text/PDF
- Verify equality using equals() and hashCode()

### UC5 - Dashboard Display
- Display employee dashboard
- Show recent payslips
- Display Year-to-Date earnings
- Stream API for data processing

### UC6 - Input Validation
- Validate Employee ID
- Validate Email
- Validate Phone Number
- Validate Password
- Custom exception handling
